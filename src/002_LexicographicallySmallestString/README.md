
## Problem

Given a string `input`, the task is to find the lexicographically smallest string that can be formed by removing at most one character from the given string. Refer [geeksforgeeks](https://www.geeksforgeeks.org/lexicographically-smallest-string-formed-by-removing-at-most-one-character/) for more details about the problem.

### example

1. input : `abcda`
   
   output : `abca`

2. input : `hello`
   
   output : `ello`

3. input : `adddd`
   
   output : `addd`
